📚 Primeira Aplicação com Spring Boot

Este projeto foi desenvolvido como parte dos meus estudos em desenvolvimento backend utilizando Spring Boot.

🚀 O que foi praticado
Criação de uma API REST simples
Implementação de endpoints GET e POST
Utilização de @PathVariable
Utilização de @RequestParam
Organização inicial utilizando Controllers e Services
Testes das requisições utilizando Postman
🎯 Objetivo

O objetivo deste projeto foi entender o fluxo básico de funcionamento de uma API, desde o recebimento das requisições até o retorno das respostas ao cliente.

🛠️ Tecnologias utilizadas
Java
Spring Boot
Maven
Postman
📖 Status do projeto

✅ Projeto desenvolvido para fins de aprendizado e prática dos conceitos fundamentais de APIs REST com Spring Boot.
